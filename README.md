# выгрузка статей из https://vk.com/@yvkurse в .csv файл

## Установка
```
git clone https://github.com/scherbininvladimir/yvkurse
cd yvkurse
pip3 install -r requirements.txt
```

## Настройка
```
В файле settings.py:
output_file - файл, в который будут выведены статьи
```

## Запуск
```
python3 get_csv.py
```
